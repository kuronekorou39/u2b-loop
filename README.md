# U2B-Loop

YouTube・ローカル動画のAB区間ループ再生プレーヤー

![U2B-Loop](assets/logo.png)

## 特徴

- **ABループ再生** - 動画の指定区間を繰り返し再生
- **YouTube対応** - URLを貼り付けるだけで再生
- **ローカルファイル対応** - PC内の動画ファイルも再生可能
- **ミリ秒単位の精度** - 0.001秒単位でAB地点を調整
- **PWA対応** - インストールしてオフラインでも使用可能
- **履歴管理** - AB区間を保存・エクスポート・インポート
- **URL共有** - AB区間を含むURLを生成して共有

## デモ

https://kuronekorou39.github.io/u2b-loop/

## 使い方

### YouTube動画
1. YouTube URLを入力欄に貼り付け
2. 「読込」ボタンをクリック
3. A地点・B地点を設定
4. 「ループ ON」で繰り返し再生

### ローカル動画
1. 「ファイル選択」で動画ファイルを選択
2. または動画ファイルを画面にドラッグ&ドロップ
3. A地点・B地点を設定
4. 「ループ ON」で繰り返し再生

## キーボードショートカット

| キー | 機能 |
|------|------|
| `Space` | 再生/一時停止 |
| `←` `→` | 5秒戻る/進む |
| `A` | 現在位置をA地点に設定 |
| `B` | 現在位置をB地点に設定 |
| `L` | ループON/OFF切替 |
| `M` | ミュート切替 |
| `H` | 左右反転 |
| `V` | 上下反転 |
| `F` | フルスクリーン |

## URL共有

AB区間を設定後、「共有」ボタンでURLをコピーできます。

```
https://kuronekorou39.github.io/u2b-loop/?v=VIDEO_ID&a=10.500&b=30.000&l=1
```

| パラメータ | 説明 |
|-----------|------|
| `v` | YouTube動画ID |
| `a` | A地点（秒） |
| `b` | B地点（秒） |
| `l` | ループ有効（1で有効） |

## インストール

### Webブラウザで使用
上記デモURLにアクセスするだけで使用できます。

### PWAとしてインストール
1. ブラウザでアプリを開く
2. アドレスバーのインストールアイコンをクリック
3. デスクトップ/ホーム画面にアプリが追加される
